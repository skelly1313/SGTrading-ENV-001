# DOCS/decisions/

ADR-style records — *why* we picked X over Y.

One file per decision, named `NNNN-short-title.md` (sequential):

```markdown
# NNNN — <title>

- **Date:** YYYY-MM-DD
- **Status:** proposed | accepted | superseded by NNNN

## Context
What forced the decision.

## Options considered
Brief enumeration.

## Decision
What we chose.

## Consequences
What this locks us into and what it leaves open.
```
